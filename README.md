# Proyecto

Este es el proyecto final del curso de React de TalentoTech de Aquino Maria Laura
Es un sitio de E-Commerce que permite Administrar los productos de la tienda (siendo un usuario Administrador) 
y permite navegar los productos de la tienda, agregarlos al carrito. Para visitar el carrito con los productos seleccionados,
se debe loguear.
Respecto a la funcionalidad, hay que aclarar que para ingresar a la "Administración de Productos" tenes que ingresar con el usuario "admin".

# Ejecutar localmente

Para poder ejecutar el codigo del proyecto localmente, pararse en la carpeta donde esta incluido el proyecto (la carpeta inicial)

/
├── public/
├── src/

Alli ejecutar el siguiente comando para que se instalen todas las dependencias necesarias:

npm install

Luego para poder compilar y que se levante en un puerto local, ejecutar lo siguiente:

npm run dev

Va a mostrar en la consola en que puerto va a poder navegarse el sitio.

Por ej:
Local:   http://localhost:5173/
